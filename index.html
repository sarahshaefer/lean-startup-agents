Lean Startup
Agent Workflow
🔍
Pain Analyst
Deep understanding of the problem space
This agent thinks like an anthropologist embedded in the customer's world. It doesn't accept surface-level pain — it digs for the real, underlying frustration people can't easily articulate.
💡
Maverick Ideator
Bold, non-obvious value propositions
This agent breaks rules. It ignores what already exists, challenges assumptions about how a solution "should" look, and proposes ideas that might initially sound crazy. The goal is not to be safe — it's to be interesting.
🧪
Hypothesis Engineer
What must be true for this to work?
This agent thinks like a scientist. It takes your value proposition and breaks it into testable beliefs — the riskiest assumptions that, if wrong, would kill the business. It prioritises ruthlessly.
🛠️
MVP Architect
The smallest thing you can build to learn
This agent is obsessed with speed and learning. It strips away everything that isn't necessary to test the core hypothesis. Its MVP might not even be software — it might be a landing page, a spreadsheet, or a manual process.
💼
VC Critic
Hard questions from a hard investor
This agent has seen 1,000 pitches and funded 10. It doesn't care about enthusiasm. It asks the questions founders avoid, pokes at market size, questions defensibility, and stress-tests every assumption. Its job is to make you better.
📡
Research Scout
Markets, competition, and feasibility
This agent is a neutral, rigorous analyst. It doesn't hype and it doesn't dismiss. It gathers facts about market size, existing competitors, and technical feasibility — and gives the other agents the grounding they need to stay honest.
🏗️
Product Architect
From insight to scalable product
This agent takes everything the other agents have uncovered and asks one question: how do we turn this into a product that works without the founder? It bridges the gap between a consulting insight and a scalable business.
